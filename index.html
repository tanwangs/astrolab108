New Week, New Topic!

Here are my learnings in the AstroLab 108 workshop.

Week 1, Basics of Python!

In this week, we explored the platform GoogleColab and Python, NumPy, and Matplotlib library. These tools helped us visualize and analyze real datas with the help of plotting graphs using data.

Week 2, Exoplanet Detection!

In this week, we explored about Exoplanets, and learned the Transit Method and Radial Velocity Method which are used to detect Exoplanets. We also learned to interpret graphs and we discussed about PANOPTES.

Week 3, Finding Exoplanets using Real Data!

This week, we learned how to find exoplanets by using real data from NASA's TESS satellite and how the satellite works using the method, Transit Detection. We also learned about the lightcurve library which helped us access the light curve of a star, which we plotted down and flattened it to find the planet's orbital period.

Week 4, Git and GitHub!

For this week, we learned about Git and GitHub, learning some basic commands. We create GitHub accounts in which we learned to upload some files into our repositories. We also explored about Operating Systems and user interfaces.

Week 5, Building CV and designing Website!

This week, we learned about making CVs and Resumes using OverLeaf, a platform where you can program documents using LaTex. We learned how it worked and differences between OverLeaf and Google Docs. As for practical, we made a CV about ourselves. On the same week, we also learned Website Creation in which we took templates from HTML5 UP! (This website is a template from HTML5). We learned about editing these templates and uploading images aswell.
